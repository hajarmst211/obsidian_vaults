## Inteligence artificielle
##### Definition
- Terme inventé par John McCarthy en 1956
- L’IA commence là où l’informatique classique s’arrête
- Simuler le comportement humain
-  automatise les taches intellectuelles pour créer des machines capable de penser comme les humains

##### Fondements:
- philosophie
- phsycologie
- mathematique
- economie
- neurosciences

#####  Historique de l'IA
- En 1943, McCulloch et Pitts proposent le modèle de neurone artificiel de qui
représente le début des travaux sur les réseaux de neurones.
- ordinateur neuronal
- un programme pour jouer aux échecs.
##### machine learning
- L'objectif de l'apprentissage automatique est de concevoir des programmes
pouvant s‘améliorer automatiquement avec l’expérience.
##### apprentissage supervise/unsupervise/semi-supervise
| Type d’apprentissage | Données utilisées                                   | Objectif                                         | Exemples                           |
| -------------------- | --------------------------------------------------- | ------------------------------------------------ | ---------------------------------- |
| Supervisé            | Données étiquetées                                  | Prédire un résultat connu                        | Régression, Classification         |
| Non supervisé        | Données non étiquetées                              | Trouver des structures cachées                   | Clustering, Réduction de dimension |
| Semi-supervisé       | Peu de données étiquetées + beaucoup non étiquetées | Améliorer les performances avec peu d’étiquettes | Apprentissage mixte                |
| Par renforcement     | Données issues d’interactions                       | Apprendre à agir pour maximiser une récompense   | Q-Learning, Deep RL                |
