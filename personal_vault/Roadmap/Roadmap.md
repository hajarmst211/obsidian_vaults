
### 1. Data Collection (The Scraper)
*The goal: Get data from the internet that doesn't offer a download button.*

1.  **Build a simple HTML Scraper:** Use Python (`BeautifulSoup`) to scrape a Wikipedia table and save it as a CSV.
2.  **Build a "Headless" Browser Bot:** Use `Selenium` or `Playwright` to log into a website, click buttons, and scrape data that is hidden behind a login.
3.  **Create an API Consumer:** Write a script that connects to the Spotify or Twitter/X API, downloads your user data, and handles "pagination" (looping through pages of results).
4.  **Build a Price Tracker:** Create a script that checks an Amazon product price every hour and logs the history to a file.
5.  **Create a Rate-Limiter:** Modify your scraper so it pauses automatically so you don't get banned by the website.

### 2. Data Plumbing (The Pipeline Builder)
*The goal: Move data automatically from Point A to Point B, cleaning it on the way.*

1.  **The "Dirty Data" Cleaner:** Download a messy dataset (bad dates, missing values, weird symbols). Write a script (Python `Pandas` or `Polars`) that cleans it and outputs a pristine file.
2.  **Build a Watchdog Script:** Write a program that "watches" a specific folder on your computer. Whenever a new file is dropped there, it automatically processes it and moves it to a "Finished" folder.
3.  **Create a Database Loader:** Write a script that reads 1,000 JSON files and inserts them efficiently into a Postgres or SQLite database.
4.  **Build an Alert System:** Write a script that scans a dataset. If "Revenue" drops below $0, it immediately sends you an email or a Discord notification via webhooks.

### 3. The Data Warehouse (The Architect)
*The goal: Structure data so it answers questions fast. This is SQL-heavy programming.*

1.  **Design a Star Schema:** Take a flat Excel sheet of "Sales" and break it into SQL tables: `Fact_Sales`, `Dim_Customer`, `Dim_Product`, `Dim_Time`.
2.  **Implement "Slowly Changing Dimensions":** Write a SQL logic that handles a customer changing their address. Do you overwrite the old one? Or do you keep a history? (Program the logic to keep history).
3.  **Build a KPI View:** Write a complex SQL query that calculates "Monthly Recurring Revenue" and "Churn Rate" from raw login logs.
4.  **Use DuckDB:** Use `DuckDB` (an in-process SQL OLAP database) to query a massive CSV file (1GB+) instantly on your laptop using Python.

### 4. Programmatic Reporting (The Automator)
*The goal: Make reports without opening a BI tool.*

1.  **The "Excel Robot":** Use Python (`openpyxl` or `xlsxwriter`) to generate a formatted Excel report from scratch. It should have formulas, bold headers, and conditional formatting, all generated by code.
2.  **PDF Generator:** Write a script that takes last week's sales and generates a distinct PDF invoice for every single client automatically (`ReportLab` or `FPDF`).
3.  **The "Briefing" Bot:** Write a script that runs every morning at 8 AM, calculates the top 3 metrics of yesterday, and sends a formatted Slack message to a channel.

### 5. Data Application (The Tool Maker)
*The goal: Build mini-apps for other people to explore data.*

1.  **Build a Streamlit App:** Use Python `Streamlit`. create a web interface where a user can upload a CSV, and your code automatically summarizes it for them on the screen.
2.  **Create a Data API:** Build a simple REST API (using `FastAPI` or `Flask`) that allows other developers to query your database. (e.g., `GET /api/sales?date=2023-01-01`).
3.  **Interactive Map:** Use `Leaflet` or `Folium` to plot GPS coordinates from a dataset onto a map programmatically.

If you want to see if you get hooked:

**Try Project 2.4 (The Alert System) or 4.1 (The Excel Robot).**
This gives you a feeling of power—you write code, and suddenly a manual office task disappears. If you get a rush of dopamine from automating a boring spreadsheet task using Python code, you will love **Analytics Engineering** or **Data Engineering**.
